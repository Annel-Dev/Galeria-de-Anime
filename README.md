# Galería de Anime

Práctica de HTML y CSS donde armé una galería simple con imágenes 
de personajes de anime que me gustan. El objetivo era practicar 
estructura básica, estilos, sombras y efectos hover.

## ¿Qué aprendí haciendo esto?

- Estructura básica de HTML (head, body, listas, imágenes)
- Estilos con CSS: padding, margin, border-radius, box-shadow
- Gradientes con linear-gradient
- Efecto hover con transition y transform
- Cómo organizar archivos (carpeta de imágenes separada del HTML)

## Tecnologías

- HTML5
- CSS3

## Notas

Este proyecto es parte de mi proceso aprendiendo desarrollo web 
desde cero. Sé que tiene cosas mejorables (semántica del HTML, 
uso de flexbox para layout responsive, variables CSS para los 
shadows repetidos) y planeo rehacerlo cuando aprenda flexbox 
y grid.
